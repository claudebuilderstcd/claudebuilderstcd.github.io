Join us for an introductory workshop on the Claude API! This hands-on session is perfect for beginners who want to start building with Claude.

## What You'll Learn

- Setting up your Anthropic API account
- Making your first API calls
- Understanding prompts and responses
- Building a simple chatbot application

## Requirements

- Laptop with Python installed
- TCD email address (for API credits)
- Curiosity and enthusiasm!

## Agenda

1. **Introduction** (15 min) - Overview of Claude and the API
2. **Setup** (15 min) - Getting your API keys and environment ready
3. **Hands-on Coding** (60 min) - Building your first application
4. **Q&A and Networking** (30 min)

Don't forget to fill out both the Interest Form and Check-in Form to receive your API credits!
